Snippet Import

Summary:

Import snippets in GlobalSight to populate the Snippet Library.  Optional

Select a File To Import

Select a Snippet Import File that contains Snippets for your Snippet Library. Refer to the Snippet Import File Example for an example of a Snippet Import File. The Snippet Import File Example works in conjunction with the GS-Tagged Source Page Example.  Use the Browse... button to select the snippet file to import.

File Encoding

The character encoding of the file you are Snippet Import File you are importing.

Buttons

 

Browse...

Locate the support file on your PC to upload.

Cancel

Cancels the snippet import.

Next

Moves to the next screen of snippet import (Testing Import).
