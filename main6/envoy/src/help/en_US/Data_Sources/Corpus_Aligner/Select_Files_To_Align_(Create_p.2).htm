Select Files To Align (Create p.2)

Summary

Select the files to be aligned.

Source File

Browse to the source files to include in the package.  Select a file by checking the box next to the file name.

Target File

Browse to the corresponding target files to include in the package.  Select a file by checking the box next to the file name.

Selected File Pairs

Displays the file pairs selected for the alignment.  Select a file pair by checking the box next to the file name.

Buttons

 

(Folder Name)

Clicking on the name of a folder opens that folder, showing files that have an extension selected in the previous screen.

Moves up a directory.

Adds the selected files to the aligner package.  You must select at least one file from both the source and target windows.

Removes the selected file pairs from the aligner package.

Cancel

Cancel aligner package creation and return to Aligner Packages.
